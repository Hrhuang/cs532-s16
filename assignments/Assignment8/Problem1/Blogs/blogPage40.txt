/**
 * jQuery Mobile Menu 
 * Turn unordered list menu into dropdown select menu
 * version 1.0(31-OCT-2011)
 * 
 * Built on top of the jQuery library
 *   http://jquery.com
 * 
 * Documentation
 * 	 http://github.com/mambows/mobilemenu
 */
(function($){
$.fn.mobileMenu = function(options) {
	
	var defaults = {
			defaultText: 'Navigate to...',
			className: 'select-menu',
			containerClass: 'select-menu-container',
			subMenuClass: 'sub-menu',
			subMenuDash: '&ndash;'
		},
		settings = $.extend( defaults, options ),
		el = $(this);
	
	this.each(function(){
		// ad class to submenu list
		el.find('ul').addClass(settings.subMenuClass);

        // Create base menu
		$('<div />',{
			'class' : settings.containerClass
		}).insertAfter( el );
        
		// Create base menu
		$('<select />',{
			'class' : settings.className
		}).appendTo( '.' + settings.containerClass );

		// Create default option
		$('<option />', {
			"value"		: '#',
			"text"		: settings.defaultText
		}).appendTo( '.' + settings.className );

		// Create select option from menu
		el.find('a').each(function(){
			var $this 	= $(this),
					optText	= '&nbsp;' + $this.text(),
					optSub	= $this.parents( '.' + settings.subMenuClass ),
					len			= optSub.length,
					dash;
			
			// if menu has sub menu
			if( $this.parents('ul').hasClass( settings.subMenuClass ) ) {
				dash = Array( len+1 ).join( settings.subMenuDash );
				optText = dash + optText;
			}

			// Now build menu and append it
			$('<option />', {
				"value"	: this.href,
				"html"	: optText,
				"selected" : (this.href == window.location.href)
			}).appendTo( '.' + settings.className );

		}); // End el.find('a').each

		// Change event on select element
		$('.' + settings.className).change(function(){
			var locations = $(this).val();
			if( locations !== '#' ) {
				window.location.href = $(this).val();
			};
		});

	}); // End this.each

	return this;

};
})(jQuery);

      $allVideos = $allVideos.not(ignoreList); // Disable FitVids on this video.

      $allVideos.each(function(){
        var $this = $(this);
        if($this.parents(ignoreList).length > 0) {
          return; // Disable FitVids on this video.
        }
        if (this.tagName.toLowerCase() === 'embed' && $this.parent('object').length || $this.parent('.fluid-width-video-wrapper').length) { return; }
        if ((!$this.css('height') && !$this.css('width')) && (isNaN($this.attr('height')) || isNaN($this.attr('width'))))
        {
          $this.attr('height', 9);
          $this.attr('width', 16);
        }
        var height = ( this.tagName.toLowerCase() === 'object' || ($this.attr('height') && !isNaN(parseInt($this.attr('height'), 10))) ) ? parseInt($this.attr('height'), 10) : $this.height(),
            width = !isNaN(parseInt($this.attr('width'), 10)) ? parseInt($this.attr('width'), 10) : $this.width(),
            aspectRatio = height / width;
        if(!$this.attr('id')){
          var videoID = 'fitvid' + Math.floor(Math.random()*999999);
          $this.attr('id', videoID);
        }
        $this.wrap('<div class="fluid-width-video-wrapper"></div>').parent('.fluid-width-video-wrapper').css('padding-top', (aspectRatio * 100)+"%");
        $this.removeAttr('height').removeAttr('width');
      });
    });
  };

<a href="http://www.thebookdesigner.com/2016/04/avast-piracy-self-publisher/comment-page-1/">This article</a> on the Book Designer is sensible and useful advice. I make my living as an author but always release DRM-free, since <a href="http://karldrinkwater.blogspot.co.uk/search/label/DRM">DRM</a> has caused me so many problems as a consumer (in films, books, and <a href="http://karldrinkwater.blogspot.co.uk/2013/11/why-i-hate-drm-i-want-to-play.html">games</a>). It only ever penalises legitimate customers. The best thing is to treat your customers well, and not worry about the ones who wouldn't pay anyway. Who knows, they might give you a good review instead - that's worth more than the cost of a book.<br />

<a href="https://1.bp.blogspot.com/-zCmNJGG2DCM/VwPDhovWhWI/AAAAAAAABZY/Iuz9mW0dEFMVxRwhyXYWJttHOo_6vtDkw/s1600/IMAG0062.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="191" src="https://1.bp.blogspot.com/-zCmNJGG2DCM/VwPDhovWhWI/AAAAAAAABZY/Iuz9mW0dEFMVxRwhyXYWJttHOo_6vtDkw/s320/IMAG0062.jpg" width="320" /></a></div>
<div style="text-align: center;">
<i>&nbsp;A view from the grounds</i></div>
<br />
As regular readers will know, I like going on <a href="http://karldrinkwater.blogspot.co.uk/2015/12/writing-courses-in-welsh-coastal-life.html">writing courses</a>. I haven't yet written anything about the Arvon Foundation course I attended 9th&#8211;14th November 2015: "<a href="http://www.arvon.org/course/fiction-work-progress-5/">Work In Progress: Writing is rewriting</a>", tutored by <a href="http://www.nickbarlay.com/">Nick Barlay</a> and <a href="http://www.diranadebayo.com/">Diran Adebayo</a> at <a href="http://www.arvon.org/centres/the-hurst/">The Hurst</a>. Now I will rectify that error.<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://3.bp.blogspot.com/-OIGeIxo07ts/VwPDnaLfHiI/AAAAAAAABZc/eBJHuSvy1603VvqlnfTQtcIiM39mdVQHg/s1600/Good%2BLife%2Bcafe%252C%2BShrewsbury%252C%2Bon%2Bway%2Bthere.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="191" src="https://3.bp.blogspot.com/-OIGeIxo07ts/VwPDnaLfHiI/AAAAAAAABZc/eBJHuSvy1603VvqlnfTQtcIiM39mdVQHg/s320/Good%2BLife%2Bcafe%252C%2BShrewsbury%252C%2Bon%2Bway%2Bthere.jpg" width="320" /></a></div>
<div style="text-align: center;">
<i>I stopped off at the <a href="http://www.goodlife-shrewsbury.co.uk/">Good Life</a> in Shrewsbury</i></div>
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://4.bp.blogspot.com/-QL_2nZ1fmpA/VwPD0G_flaI/AAAAAAAABZg/xOnvKZ5HIh8YUxlTjZ74cFrR4vnr5lFmA/s1600/IMAG0058.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="191" src="https://4.bp.blogspot.com/-QL_2nZ1fmpA/VwPD0G_flaI/AAAAAAAABZg/xOnvKZ5HIh8YUxlTjZ74cFrR4vnr5lFmA/s320/IMAG0058.jpg" width="320" /></a></div>
<div style="text-align: center;">
<i>Lying train people! My train was the <br />16:50 Carmarthen - note that it still hadn't arrived by 17:07!</i></div>
<br />
After a lengthy journey (anywhere from Aberystwyth by train is a lengthy journey) I met up with some other participants and we got a taxi to The Hurst for welcome talks, getting to know each other, and a big meal.I had a lovely large bedroom, set slightly away from the main building: yes, I had warned them in advance that I would be bringing my guitar.<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://3.bp.blogspot.com/-08uzpdOgePU/VwPEJx_F6VI/AAAAAAAABZo/aqtWHa2Bi0kqGgeng-wdxLN2N752e6MfA/s1600/IMAG0059.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="191" src="https://3.bp.blogspot.com/-08uzpdOgePU/VwPEJx_F6VI/AAAAAAAABZo/aqtWHa2Bi0kqGgeng-wdxLN2N752e6MfA/s320/IMAG0059.jpg" width="320" /></a></div>
<div style="text-align: center;">
<i>Morning run</i></div>
<br />
The teaching format for the week was one long session each day, rather than two shorter ones as has been the case at some other courses I attended; but this freed up more time for tutorials, which was necessary when there were lots of attendees and we got two tutorials each, one with each tutor. We sat around a large round table which was great for the main sessions, you could see everyone easily, and all felt able to contribute. The sessions were a mix of discussions, examining examples, and creative exercises. One thing here: my favourite format after a short creative exercise is when everyone gets a chance to read out their work or pass, moving around the table. I prefer that to the system where the group are asked if anyone would like to read out their work (with only time for a few examples), because in that case the most 
assertive characters get more opportunity to read than the less pushy ones; 
it also encourages leaping forward to volunteer otherwise you won&#8217;t get the chance. On the other hand with such a large group as the Arvon course hosted it isn't possible for everyone to read out every time; I just wonder if there could be a fairer system, such as recording how many times people read out, and making sure everyone has the same number of opportunities.<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://3.bp.blogspot.com/-yVtsDCDf0OI/VwPEV8M1M8I/AAAAAAAABZw/lTW6O1JvtIUVtR3uUDqDgt4OSRCV6ziXg/s1600/IMAG0070.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="191" src="https://3.bp.blogspot.com/-yVtsDCDf0OI/VwPEV8M1M8I/AAAAAAAABZw/lTW6O1JvtIUVtR3uUDqDgt4OSRCV6ziXg/s320/IMAG0070.jpg" width="320" /></a></div>
<div style="text-align: center;">
<i>I wandered lonely as a leaf</i></div>
<br />
I realised I'd never been on a course with male tutors before. I've always been lucky in going on good quality courses with excellent tutors, and that didn't change. Nick and Diran made a good team, whilst also being individual in their approaches. Nick seems fiercely serious at first (but is 
really friendly!); he normally gave direct advice, in concise points, very clear and considered with incisive perceptions, and a welcome willingness to be open to other interpretations of work. Diran preferred to feel his way around a 
subject with many examples and enjoyable animated anecdotes, giving close 
thought to the detail of a work whilst being humble and self-deprecating. Both are very intelligent people, helpful, friendly, 
knowledgeable. Both also broadened our appreciation via many samples 
of other writers, different approaches to issues, different styles, 
which helped make the advice concrete. I'm grateful to both of them. Teaching is 
hard. Reading, work, preparation, tutorials, support, professionalism, being on duty 
all week. It was a really good course.<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://1.bp.blogspot.com/-pehjMMBcu98/VwPEtUwv6rI/AAAAAAAABaA/PXktqmmftGoBxdVGVOz5pOc-SFTtxIydw/s1600/IMAG0072.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="320" src="https://1.bp.blogspot.com/-pehjMMBcu98/VwPEtUwv6rI/AAAAAAAABaA/PXktqmmftGoBxdVGVOz5pOc-SFTtxIydw/s320/IMAG0072.jpg" width="191" /></a></div>
<div style="text-align: center;">
<i>We stand on the shoulders of giants.&nbsp;</i></div>
<div style="text-align: center;">
<i>Or grassy banks, one of the two.</i></div>
<br />
By the end of the week I had seen subtle yet 
noticeable increases in confidence and technique in all participants. The final night&#8217;s readings are something that can create trepidation &#8211; obviously
 for any readers nervous about performance or the quality of their work, but
 also in listeners if faced with hours of readings that require 
concentration and patience. In this case all of us were surprised that the readings flew by, 
with interesting stories, varied voices and topics, tones and deliveries ranging from
 dark (e.g. my contribution) to comic (which had everyone laughing). It really was impressive and I genuinely enjoyed the work of every other 
participant. It's so good to have a week with such talented writers and tutors.<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://3.bp.blogspot.com/-qe-7ro9xXZA/VwPE4sU-ZiI/AAAAAAAABaI/Q9zyppfzfbw1wqXXTbou3flfPQYnAXfUw/s1600/IMAG0067.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="265" src="https://3.bp.blogspot.com/-qe-7ro9xXZA/VwPE4sU-ZiI/AAAAAAAABaI/Q9zyppfzfbw1wqXXTbou3flfPQYnAXfUw/s320/IMAG0067.jpg" width="320" /></a></div>
<div style="text-align: center;">
<i>I'm a little pixie.</i></div>
<div style="text-align: center;">
<i>Hear me roar.</i> </div>
<br />
Socially we were a great group too. Supportive and friendly (some of us set up an online group where we could continue to critique each others work after the course ended). I would be doing guitar practice in my room and someone would turn up: "Hey Karl, sorry to interrupt, but I thought your comments earlier were really helpful, any chance you'd be willing to have a read of this and help me out?" Which I always did, meaning discussion of writing didn't end with the evening meal. Feedback and constructive criticism is something we all benefit from.<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://4.bp.blogspot.com/-21MSPWb0spU/VwPElcJMn9I/AAAAAAAABZ4/yvy0EnUtM5M5O18MJoBrVlZVrkXhr_nhw/s1600/IMAG0077.jpg" imageanchor="1" style="margin-left: 1em; margin-right: 1em;"><img border="0" height="191" src="https://4.bp.blogspot.com/-21MSPWb0spU/VwPElcJMn9I/AAAAAAAABZ4/yvy0EnUtM5M5O18MJoBrVlZVrkXhr_nhw/s320/IMAG0077.jpg" width="320" /></a></div>
<div style="text-align: center;">
<i>The view from my room</i></div>
<br />
I
 was surprised that despite only two hours sleep on some of the nights 
(nothing dodgy, just a combination of staying up late in the lounge 
drinking and talking about books; my head buzzing with thoughts; 
sleeping in a strange room; wind in the trees; and lacewings landing on 
my face and waking me up, asking to be let out the front door) I never flaked out, because the people and materials were interesting and 
inspiring. I 
generally got up at 7am, and went for a pre-breakfast run to watch the world 
wake up. Always the best way to start the day, though I only tend to do it when I am away from home!<br />
<br />
One downside to my visit was that the centre use poisons on site to kill wildlife, most commonly large traps aimed at poisoning rats. <a href="http://www.nbc-2.com/story/31560779/rat-poison-on-sanibel-killing-wildlife-up-the-food-chain#.VwPAtFJghuo">That's a horrible system, since many animals eat the same food</a>, and there's no way to make it available to rats without it also being accessible to mice, voles, 
hedgehogs, squirrels, frogs, slugs etc. Some of those can, in turn, be eaten (and poison) cats, <a href="http://www.environmentalhealthnews.org/ehs/news/rodenticides-kill-wildlife">birds of prey</a>, hedgehogs, foxes 
etc. (By the way, if you've ever seen poison kill something, you'll know how gruesome it is; other traps are just as cruel, such as <a href="https://www.youtube.com/watch?v=TndSWlEhuwc">glue traps where the animals starve to death</a>, and gnaw off their own limbs to try and escape). It's also pretty ineffective - most rats don't eat the poison; some survive and become immune; and studies have proved that new rats from outside quickly replace dead rats. I prefer prevention to cure. Instead of traps to get rid of the 
creatures we share the land with, it is better to make sure buildings are secure and well-maintained, and food is not accessible or wasted: then you have no issue. As well as being cruel, traps with 
dead things in are not hygienic (there were traps in the kitchen too). The fantasist in me would suggest that having poisons on site when highly-strung writers have been critiquing each other, then cook group meals, may not be a good idea (fuel for a horror or crime story there.) The Hurst has a "Sustainability And Conservation" policy, which includes comments like <i>"Arvon is committed to environmental sustainability [...] Our objectives are - to minimise our impact on the natural environment"</i>: as such, poisons and traps and killing seem somewhat hypocritical.<br />
<br />
Despite that disappointing caveat, I did enjoy the week. <br />
<br />
+ Helpful centre staff<br />
+ See both tutors one-on-one<br />
+ Writers introduced new styles<br />
+ Great morning run<br />
+ Large room with a good view in front of the desk<br />
+ Food &#8211; lots, with fresh fruit, no problem being a vegan<br />
+ Warm, even in winter<br />
<br />
- A rule that you have to sign out when you leave the building (which I ignore on principle)<br />
- Animal traps/poisons<br />
<br />
+/- No Internet - good or bad depending on your life. I had no problems with that.<br />
+/- Cooking rota. Some people love the communal aspect of this (I always have fun cooking with my fellow writers; we tend to get a bit sloshed) but some people don't enjoy group cooking.<br />
<br />

<br />
Over on the ALLi website you'll find <a href="http://selfpublishingadvice.org/dy-writing-and-editing-tools-for-indie-authors/">an article where I talk about useful tools for writers</a> (an updated version of <a href="http://karldrinkwater.blogspot.co.uk/2016/02/editing-tools-for-writers.html">this post</a>).<br />
<br />
Just in case that post ever disappears, here's a backup of the contents.<br />
<br />
<hr />
<br />
<span style="font-size: large;">ANALYSIS TOOLS FOR WRITERS</span><br />
Intuition and human experience combine in successful writing and editing. However, just as chess software can analyse thousands of options per second, software can also analyse thousands of words quicker and more consistently than a human. We want our books to be as good as they can be, so the canny writer makes use of&nbsp; whatever tools work for them to create a bibliographic arsenal.<br />
<br />
MY WRITING PROCESS<br />
<ul>
<li>First draft banged out while in the flow of the story: just make up details, leave errors in &#8211; it's best to keep going and stay within the fictional world for as long as possible.</li>
<li>Edit, rewrite, edit. Use software tools (next section) to help with the rewriting &#8211; they help me spot common errors I make, and fix them myself &#8211; issues such as word repetition, redundancy, consistency. And in that process I learn to avoid making those errors in the future.</li>
<li>Read the work aloud (at least the dialogue), or get software/Kindle to do it for me. Fix awkward phrasing.</li>
<li>Once it's all done I then run it past editors, beta readers, proofreaders, or whoever else provides the vital human input.</li>
</ul>
During a discussion with some other writers recently I found out that they didn't know writing analysis tools existed, so here&#8217;s a summary of the ones I&#8217;m aware of.<br />
<br />
<span style="font-size: large;">THE TOOLS</span><br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://2.bp.blogspot.com/-d010PdTvtLA/VrY3-zRVcVI/AAAAAAAABN8/_pNU6d28uPE/s1600/Image3.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;"><img border="0" height="177" src="https://2.bp.blogspot.com/-d010PdTvtLA/VrY3-zRVcVI/AAAAAAAABN8/_pNU6d28uPE/s320/Image3.jpg" width="320" /></a></div>
<a href="http://editminion.com/">EDITMINION</a> <br />
Free! Paste text into the box, choose which things you want it to report on (I tend to leave it on the defaults), and click "Edit". You'll get a summary analysis, then when you scroll down you'll see your text with some of the words highlighted &#8211; hover over them to find out why (the colours should match the summaries at the bottom right of the browser). The main things EditMinion points out are: adverbs to consider removing; weak words: homonyms; prepositions at the end of a sentence; passive voice; clichs.<br />
<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://4.bp.blogspot.com/-dCHvacTAB9o/VrY6DyDEwLI/AAAAAAAABOI/0w155qL852c/s1600/Image4.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;"><img border="0" height="169" src="https://4.bp.blogspot.com/-dCHvacTAB9o/VrY6DyDEwLI/AAAAAAAABOI/0w155qL852c/s320/Image4.jpg" width="320" /></a></div>
<a href="http://www.hemingwayapp.com/">HEMINGWAY APP</a><br />
Free! Select the sample text, delete it, paste in your own. The tool is live, immediately reporting on what you've pasted or typed. It will point out sentences which are complex and you might want to simplify; and like EditMinion, it will highlight passive voice and adverbs. A nice extra feature is a readability grade, and Hemingway App is also good for flagging up phrases with simpler alternatives.<br />
<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://3.bp.blogspot.com/-tnogB1zMDkY/VrY7J5O07VI/AAAAAAAABOQ/fZq8FvJJJdE/s1600/Image5.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;"><img border="0" height="195" src="https://3.bp.blogspot.com/-tnogB1zMDkY/VrY7J5O07VI/AAAAAAAABOQ/fZq8FvJJJdE/s320/Image5.jpg" width="320" /></a></div>
<a href="http://www.proz.com/PerfectIt/Consistency_Checker">ONLINE CONSISTENCY CHECKER</a><br />
As the name suggests, this is purely for consistency &#8211; it does not do the checks mentioned in the tools above, so is a useful addition rather than a replacement. Upload a document; download their report. It will flag up things like inconsistent spelling and hyphenation.<br />
<br />
<br />
<br />
<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://3.bp.blogspot.com/-JgpL1V7A1Wk/VwJv9YcfiWI/AAAAAAAABYw/4aiKgz5Xf6s7i-t0b5s5fZ4mBo7mpF0eg/s1600/autocrit%2Bin%2Baction.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;"><img border="0" height="161" src="https://3.bp.blogspot.com/-JgpL1V7A1Wk/VwJv9YcfiWI/AAAAAAAABYw/4aiKgz5Xf6s7i-t0b5s5fZ4mBo7mpF0eg/s320/autocrit%2Bin%2Baction.jpg" width="320" /></a></div>
&nbsp;<a href="https://www.autocrit.com/">AUTOCRIT</a><br />
The full version requires an account and subscription fee. You can get an idea of what it does without paying, or even ask them to set you up with a free trial. AutoCrit will give you summaries and statistics about the work, a visual guide to sentence length, commonly used dialogue tags, how they think your work compares to published fiction on various metrics, passive voice indications, warnings of possible clichs, your most commonly used or repeated words and phrases, plus my personal favourite report: unnecessary filler words.<br />
<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://3.bp.blogspot.com/-MFN1BIiqGaw/VwJwBjLgVxI/AAAAAAAABY0/PW_bj0AeB34ltpTPBJVDnvf6GMcrhNzHg/s1600/prowritingaid%2Bin%2Baction.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;"><img border="0" height="193" src="https://3.bp.blogspot.com/-MFN1BIiqGaw/VwJwBjLgVxI/AAAAAAAABY0/PW_bj0AeB34ltpTPBJVDnvf6GMcrhNzHg/s320/prowritingaid%2Bin%2Baction.jpg" width="320" /></a></div>
&nbsp;<a href="https://prowritingaid.com/">PROWRITINGAID</a><br />
Another subscription tool, though not expensive &#8211; a lifetime subscription to ProWritingAid is only $120 (c. &#163;83) which is less than a year's subscription to AutoCrit. Again, you can <a href="https://prowritingaid.com/en/App/RequestTrial">request a trial</a> and get a good idea of what it does that way. It gives in-depth reports on over-used words, sentence length, grammar issues, vague or abstract words and so on.<br />
<br />
<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://2.bp.blogspot.com/-BRC1jYeq_eM/VrZB5umviHI/AAAAAAAABO0/R6m4iCSyHlQ/s1600/Image2.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;"><img border="0" height="179" src="https://2.bp.blogspot.com/-BRC1jYeq_eM/VrZB5umviHI/AAAAAAAABO0/R6m4iCSyHlQ/s320/Image2.jpg" width="320" /></a></div>
<a href="http://www.wordrake.com/">WORDRAKE</a><br />
A paid tool. This tool scrapes redundant words out, making suggestions for replacements, so you have shorter, tighter, clearer language use. Although not aimed primarily at fiction writers it has obvious uses. It's not cheap &#8211; the plugin for Word would be $129 (c. &#163;89) a year, with reductions for subscribing for multiple years. That's quite a lot for a tool that only serves a single purpose. I like the fact that it ran in my word processor with its own tab, so no need to upload and analyse documents online, though installing WordRake was problematic on my PC. I only used about a quarter of the suggestions WordRake made, but they were good ones. <br />
<br />
<br />
<div class="separator" style="clear: both; text-align: center;">
<a href="https://1.bp.blogspot.com/-vDYjhnp6U4A/VrZBkgbDhpI/AAAAAAAABOw/gI4eZvWNnaM/s1600/Image1.jpg" imageanchor="1" style="clear: left; float: left; margin-bottom: 1em; margin-right: 1em;"><img border="0" height="170" src="https://1.bp.blogspot.com/-vDYjhnp6U4A/VrZBkgbDhpI/AAAAAAAABOw/gI4eZvWNnaM/s320/Image1.jpg" width="320" /></a></div>
<a href="https://www.grammarly.com/">GRAMMARLY</a><br />
I haven't tried this but some writers are fans. Supposedly like an upgrade to Word&#8217;s grammar checker.<br />
<br />
<br />
<br />
<br />
<br />
<br />
<span style="font-size: large;">WHAT NEXT?</span><br />
Tools are not a cure-all, and amongst the useful suggestions there will be many false positives. Tools are also not a replacement for the human touch. However, they invariably provide some useful insight. Why not try some of them out on your current or past works and see what they flag up? As indie authors it's easy for us to edit and upload improved versions of our books at any time.<br />
<br />
<span style="font-size: large;">TIPS</span><br />
<ul>
<li>If the tool is a paid one, think about possible usage patterns. For example, because of the cost of AutoCrit I don&#8217;t plan on having a permanent subscription, but whenever I finish a new book I&#8217;ll probably subscribe for a month and run my text through it. That&#8217;s the most cost-effective way for me.</li>
<li>Make use of the free trials that are available, and see if they make any suggestions on your work that are useful. We learn from many sources as we grow as writers.</li>
<li>Over time I&#8217;d predict that these tools would find fewer errors in your work, since you'd have learnt from issues flagged up in the past. Another reason to think about usage patterns when considering a subscription &#8211; as years go by you might find less and less use for some of these if you are a reflective writer.</li>
<li>Even the free triaIs were useful: I made a note of repeated errors on my part in my revision checklist document. I can then check for some of those myself in future without an external tool (or in conjunction with CTRL-F in Word).</li>
</ul>

<br />
<span style="font-size: large;"><b>Mar 29<br />Virtual Book Club: Karl Drinkwater</b></span><br />
<br />
Today, I&#8217;m delighted to welcome Karl Drinkwater to Virtual Book Club,
 an interview series in which I put questions to authors about the books
 they would like to persuade your book club to read. If you want to pose
 a question, you&#8217;ll have the opportunity to do so at the end. There is 
also a fantastic giveaway, so don&#8217;t forget to enter!<br />

Karl is originally from Manchester but has lived in Wales for over 
fifteen years, ever since he went there to do a Master&#8217;s degree: it was 
easier to stay than to catch a train back. His longest career was in 
librarianship (twenty-five years); his shortest was industrial welding 
(one week).<br />

He started writing stories when he was nine, and hasn&#8217;t stopped. His 
writing sometimes spends time in the sunlit patches of literary fiction,
 where it likes to picnic beneath an old oak tree, accompanied by a 
bottle of wine, some cake, and soul-searching peace. At other times his 
words slope off into the dark and tense shadows of horror fiction, and 
if you follow them you might hear chains rattling behind locked doors 
and the paranoid screams of the lost echoing in the distance. There is 
no obligation to enjoy both of those avenues. His aim is to tell a good 
story, regardless of genre, but it always comes down to life, death, and
 connection.<br />

When he isn&#8217;t writing or editing he loves exercise, computer games, 
board games, the natural environment, animals, social justice, and 
zombies; not necessarily in that order.<br />

<h3>
Q: Karl, perhaps you can start by telling us how you came to be a writer.</h3>
I grew up in Manchester, a place I love and hate. My father died when
 I was a child and we moved around a lot. New schools, new houses. I 
found it hard to keep making new friends so retreated into books and 
fell in love with story-telling. There were other worlds I could visit 
as I read at the top of the weeping willow, or underneath my bed by 
torchlight. Then I discovered that I could make my own worlds too and 
began writing stories and poems, often quite dark juvenilia. They were 
published in school magazines and that made me happy.<br />

With the exception of English, where I always got top marks, I was a 
rebellious and problematic child, unhappy and worried about my family. I
 experienced my first bouts of depression, a mental state that has been 
my occasional companion in life. Then I went to a further education 
college and it turned my life around. I learned to love studying &#8211; not 
because I was told to, and punished if I didn&#8217;t, but because I <em>wanted to learn</em>.
 I studied extra subjects and went to night school too, despite it 
meaning I was sometimes in college for twelve hours. I became the first 
person in my family to go to university and eventually got first class 
honours in English and Classics (specialising in Ancient Greek 
language).<br />

I moved to Wales, achieved an MSc in Information Studies, and became a university librarian.<br />

All this time I kept writing. And every year I would look back on my 
older work and see its flaws, learn from them, and continue writing new 
stories. Around ten years ago I wrote my first novel in lunchtimes while
 working full time, the only way I could fit it in around all my other 
commitments. I now have three novels to my name and have left the 
library job (after twenty-five years) to immerse myself in what I love: 
writing.<br />

<img alt="Karl as librarian" class="aligncenter size-medium wp-image-12271" height="179" src="http://jane-davis.co.uk/wp-content/uploads/2016/03/Karl-as-librarian-300x179.jpg" width="300" />
<h3>
Q: What is it about writing that means you&#8217;d leave a steady job?</h3>
Writing is something I have to do. It&#8217;s a compulsion, a desire, a way
 of communicating, of getting things out of myself, of trying to create 
something of value to other people. That drive keeps me going on days 
when the words don&#8217;t flow.<br />

<h3>
Q: What kind of thing do you write?</h3>
I do what authors are told not to do &#8211; switch between genres. Some of
 my work is literary/contemporary, stories about life, death, love and 
relationships. Other stories are horror. Yet I see the difference as 
just being one of tone. The elements I focus on are the same. Your best 
writing will always be writing you love and stories you feel compelled 
to tell. Regardless of what niche a particular work falls into my aim is
 always to tell a good story.<br />

<img alt="2000 Tunes cover" class="aligncenter size-medium wp-image-12272" height="300" src="http://jane-davis.co.uk/wp-content/uploads/2016/03/2000-Tunes-cover-212x300.jpg" width="212" />
<h3>
Q: The novel we&#8217;re going to focus on today is <a href="http://karldrinkwater.blogspot.com/p/2000-tunes.html" target="_blank">2000 Tunes</a>,
 and it&#8217;s a&nbsp;literary/contemporary novel- which is how I describe my own 
fiction.&nbsp;Where is your book set and how did you decide on its setting?</h3>
It&#8217;s set in Manchester, and a key subplot is about the two 
protagonists coming to realise that Manchester isn&#8217;t the place they once
 thought it was. Sometimes reality pushes through the dreams and you 
decide it is time to move on, an obvious metaphor for growing up. The 
other element of the setting is that it&#8217;s based in the year 2000. A time
 when people thought the world might suddenly change for the better. We 
all know how <em>that</em> worked out. But it&#8217;s an interesting liminal 
time, totally appropriate for a coming of age story about a 
music-obsessed Manchester man and an over-indulging Welsh woman choosing
 reality over the dream.<br />

<em>2000 Tunes</em> is a partner to my previous novel, <em><a href="http://karldrinkwater.blogspot.com/p/cold-fusion-2000.html" target="_blank">Cold Fusion 2000</a></em>,
 which was also set in Manchester in 2000. The two novels have similar 
themes, crossover locations, and nerdy male characters with their own 
obsessions (<em>Cold Fusion 2000</em>&#8217;s Alex was an ber-geek FE 
lecturer obsessed with physics and poetry). I think these two novels 
were getting something out of my system, purging me of my mixed feelings
 about the place where I grew up.<br />

<h3>
Q: Tell us a little about the major areas you had to research.</h3>
Location, location, location. Since both novels were set in a very 
real place I wanted to reflect that, and show how the geography of an 
area affects our perception of it. The difficulty was that the city 
centre had changed a lot in the last fifteen years. Many of the places 
in the novel have already been lost, renamed, altered or closed. <em>2000 Tunes</em>
 opens outside The Haienda, one of the world&#8217;s most famous nightclubs: 
just before it was demolished for luxury flats. I had to combine my 
memories of the city at the time with archival photos and discussions; 
my diaries were useful too. I built the city back up as it used to be 
and then let the characters breathe into that space.<br />

<h3>
Q: Do you feel under pressure to make your main characters likeable?</h3>
That&#8217;s an interesting question. I primarily try to write characters that are <em>believable</em>.
 That means they have to be flawed in some ways. But the flaws can&#8217;t be 
so repugnant that the reader loses sympathy; and if a story is a journey
 then they have to change during that journey. I remember one reviewer 
saying they hated Alex (the protagonist of <em>Cold Fusion 2000</em>) at
 the start of the novel &#8211; Alex is a self-obsessed, pretentious nerd who 
is unable to communicate with anyone properly (it was an irresistible 
irony for me to make him a teacher). Yet by the end they understood him 
and felt sympathy, even if he never became fully likeable. It&#8217;s that mix
 of qualities that makes someone believable. Samantha (the Welsh woman 
from <em>2000 Tunes</em>) makes bad choices and is prone to addictive 
behaviours, but no-one can say she doesn&#8217;t have heart and spirit. Even 
in my first novel, the survival horror <em><a href="http://karldrinkwater.blogspot.com/p/turner.html" target="_blank">Turner</a></em>,
 the main character has that mix &#8211; some selfishness, anger and a foul 
mouth that repels people; yet also the strength to defend others and the
 willingness to risk himself to do it. Even anti-heroes have to be 
likeable in some way, or the reader will turn off.<br />

<h3>
Q: What&#8217;s your favourite / least favourite aspect of your writing life?</h3>
Favourite: being at home with the cat, then becoming lost in a story 
or scene; the rare occasions where it comes to you so vividly that you 
are <em>really there</em>.<br />

Least favourite: rewriting. There is none of the excitement of 
discovery of a new work; none of the near-completion feeling that comes 
with final edits. It is entering the Bog Of Eternal Trudging, unaware of
 when you will emerge.<br />

When I wrote my first novel I decided the end result wasn&#8217;t quite 
good enough so I put it aside. I came back to it years later and 
entirely re-wrote it (even switching from first person to third person).
 Although I had really wanted to give up and move on, I knew I would 
learn more by fixing something that wasn&#8217;t working. Every element was 
inter-connected: altering a character meant adjusting the plot; removing
 a scene meant adding foreshadowing elsewhere. After separating the 
engine&#8217;s parts and reconnecting them &#8211; and replacing and polishing and 
refining &#8211; I learnt a huge amount, including how to avoid making those 
mistakes again in the future, and how to cut words without regret! But 
was it fun? Generally not.<br />

<h3>
Q: That&#8217;s really interesting. I have done exactly the same with one 
of my novels. There is no simple change. Each one has to be carefully 
woven through to see what impact it has on the rest.</h3>
<img alt="Cold Fusion 2000 cover" class="aligncenter size-medium wp-image-12273" height="300" src="http://jane-davis.co.uk/wp-content/uploads/2016/03/Cold-Fusion-2000-cover-212x300.jpg" width="212" />
<h3>
Can I ask, what are you working on at the moment?</h3>
A few collections of short stories. Over the years I&#8217;ve written a lot
 of shorter works, sometimes published, sometimes just lurking on my 
hard drive where they breed at a prolific rate in the gap between 
folders. In 2015 one of my pieces was included in an anthology of UK 
short stories, <em><a href="http://www.amazon.co.uk/Secondary-Character-Other-Stories-Susie/dp/1904958583" target="_blank">Secondary Character</a>.</em>&nbsp;The
 anthology was a great way of bringing writers with various styles and 
interests together, but illustrating that we all had the same goal &#8211; to 
communicate something to our readers. Different paths to the same place.
 It also reminded me that people do love short stories, which can offer 
great variety between two covers, a kind of snack between meals.<br />

The other factor in my decisions to put my new novels on hold was 
NaNoWriMo (National Novel Writing Month). I took part for the first time
 in 2015, and although it was tremendously hard work I succeeded in 
writing over 50,000 words in November. It involved dedication, 
endurance, and late nights, but the sheer pleasure and excitement chased
 away tiredness. The intensity generated new ideas, the pressure pushed 
me to work harder. I&#8217;m currently editing the stories, then they&#8217;ll go on
 to beta-readers, then my literary editor and proofreader. If all the 
feedback is good they&#8217;ll be available this year.<br />

<h3>
Q: So when is a short story too long?</h3>
Like any work, if it has unnecessary padding, it is too long. The end
 goal is to be read, and for our readers to gain something positive from
 the experience. The tighter you wring it, the leaner it is, the more 
that lovely muscle tone shows through.<br />

<h3>
Q: What do you do when you&#8217;re not writing? Any hobbies or party tricks?</h3>
I&#8217;m learning to play the guitar, and most weekends get together with 
friends for a drink and to play music. One plays drums, one sings, two 
share bass duty. Our band is called Organic Apocalypse and I&#8217;m hoping 
we&#8217;ll do our first live performance this year.<br />

I also like exercise, running, bits of yoga. I play computer games 
and board games. I read all the time. I did pole dancing for a while but
 decided that I didn&#8217;t look respectable enough in hot pants. I spend a 
lot of time worrying about the world, the environment, population, 
wildlife, and how humans should be treating the planet where we&#8217;re 
guests. That may make me a typical vegan.<br />
<br />

<img alt="Karl running" class="aligncenter size-medium wp-image-12274" height="272" src="http://jane-davis.co.uk/wp-content/uploads/2016/03/Karl-running-300x272.jpg" width="300" />
<h3>
Q: Names in a hat. Karl, you&#8217;ve very kindly provided some fantastic prizes for a giveaway.</h3>
<ul>
<li>1 x music prize &#8211; since 2000 Tunes is about a Manchester music 
obsessive the prize is connected to his favourite bands: Ian Brown&#8217;s 
Unfinished Monkey Business (CD); The Stone Roses Second Coming (CD). (UK
 entrants only)</li>
<li>1 x print copy of 2000 Tunes to one winner (UK only)</li>
<li>1 x print copy of Cold Fusion 2000 to another winner (UK only)</li>
<li>2 x e-books pairs of 2000 Tunes AND Cold Fusion 2000 (international)</li>
</ul>
All readers need do is leave a comment. Rather than first first 
served, I think everyone should have a chance at winning, so we&#8217;re going
 to put the names in a hat and have a grand draw. The winners will be 
notified by email.<br />

<h3>
Q: Where can we find out more about you and your work?</h3>
My <a href="http://karldrinkwater.blogspot.co.uk/" target="_blank">blog</a>
 is the primary place where I chat and muse (editor&#8217;s note: this is also
 where you&#8217;ll find links to buy the books).&nbsp;I can also be found on <a href="http://www.facebook.com/karlzdrinkwater" target="_blank">Facebook</a>, <a href="http://twitter.com/karldrinkwater" target="_blank">Twitter</a> &nbsp;and <a href="http://www.goodreads.com/author/show/5766025.Karl_Drinkwater" target="_blank">Goodreads</a>, and welcome contact from my readers in whichever medium they use.</div>
