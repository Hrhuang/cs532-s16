        String.prototype.slugify = function () {
            return this.toString().toLowerCase()

                    .replace(/[^\w\-]+/g, '')       // Remove all non-word chars
                    .replace(/\-\-+/g, '-')         // Replace multiple - with single -

                if (timer !== null) {
                    clearTimeout(timer);
                }
                timer = setTimeout(function () {
                    if (ajaxRequest !== null) {
                        ajaxRequest.abort();

                                                            <h2 itemprop="description" class="contIngredients">Ça y est, la candidature de <a class="actu" href="http://actualites.leparisien.fr/paris.html">Paris</a> aux <a target="_blank" href="http://www.leparisien.fr/sports/jo-2024">Jeux Olympiques de 2024</a> s'est officiellement dotée d'un logo. Imaginé par l'agence de design Dragon Rouge, qui avait notamment redessiné l'identité visuelle du PSG, il a été projeté ce mardi soir, à 20 h 24, sur l'Arc-de-Triomphe.</h2>
                            
                            
                            
                            <div itemprop="articleBody">
                                <p> <br />
<br />
<div class="p402_hide"><iframe frameborder="0" width="545" height="307" src="http://www.dailymotion.com/embed/video/x3r7lvl?syndication=111791&logo=0&info=0&quality=720" allowfullscreen></iframe></div><br />
<br />
Pour son lancement, les porteurs du dossier voulaient envoyer un message fort et ont donc choisi un monument parisien symbolique. Mais c'est un autre monument ô combien célèbre, la Tour Eiffel, qui symbolisera la candidature parisienne. Sur le logo, celle-ci est formée grâce aux chiffres 2 et 4 (24 pour <a class="actu" href="http://actualites.leparisien.fr/paris.html">Paris</a> 2024 mais aussi pour pour la date anniversaire du centenaire des derniers Jeux Olympiques organisés à <a class="actu" href="http://actualites.leparisien.fr/paris.html">Paris</a>, en 1924). <br />
<br />
Côté couleurs, celles «du monde et celles de l'olympisme» (partie inférieure du logo) embrassent «celles de la France» (partie supérieure).<br />
<br />
Pour Dragon Rouge, «l’idée de départ était de créer un signe de ralliement, un mouvement collectif qui fédère et mobilise l’ensemble des Français». « L’identité visuelle de Paris 2024 reflète magnifiquement les valeurs et l’ambition de la candidature, souligne Bernard Lapasset, coprésident du comité d'organisation. Elle se distingue par sa pertinence, sa modernité, son audace et son élégance.» <br />
<br />
<blockquote class="twitter-tweet" data-lang="fr"><p lang="fr" dir="ltr"><a href="https://twitter.com/hashtag/Paris2024?src=hash">#Paris2024</a> a son logo. Affiché sur l&#39;Arc de Triomphe <a href="https://twitter.com/hashtag/JO2024?src=hash">#JO2024</a> <a href="https://t.co/RYq7LlhaU9">pic.twitter.com/RYq7LlhaU9</a></p>&mdash; Bruno Fraioli (@Bruno_Fraioli) <a href="https://twitter.com/Bruno_Fraioli/status/697139955387203585">9 Février 2016</a></blockquote><br />
<script async src="//platform.twitter.com/widgets.js" charset="utf-8"></script><br />
<br />
<strong style="font-size:20px;"><strong>«Un logo facile à reproduire»</strong></strong><br />
<br />
Les porteurs du dossier avaient également souhaité un logo facile à reproduire afin que tout le monde, et notamment la jeune génération, puisse le dessiner et ainsi se l'approprier. <br />
<br />
« L’enjeu était important, rappelle Tony Estanguet, l'autre coprésident, car ce logo devait à la fois porter des valeurs fortes, tout en restant très accessible.»  <br />
<br />
« Ce logo offre l’image d’une candidature ouverte, universelle, cosmopolite, heureuse d’accueillir le monde, de célébrer l’Olympisme, désireuse de créer du lien entre les Français, et avec le reste du monde. », s'enthousiasme Denis Masseglia, patron du comité olympique français.<br />
<br />
Le logo accompagnera la candidature parisienne jusqu'au 13 septembre 2017, date à laquelle sera désignée la ville hôte.<br />

        <script>
        
        // On exécute la fonction deux fois : 
        // - une fois après que le dom soit chargé pour le redimensionné à la taille
        // - une seconde fois une fois que la page est totalement chargé pour redimmensionné à nouveau au cas où une nouvelle pub apparaitrait
        $(document).ready(function(){
            window.setTimeout(resizeBloc, 700);

            $bloc300Liste = $("#blocResize");

            $onglets = $bloc300Liste.find(".onglets");
            $content = $onglets.siblings('.content');

            $onglets.children().each(function(index){
                $(this).addClass("onglet-" + index);
            });

            $onglets.children().click(function(e){
                e.preventDefault();
                this_class = $(this).attr("class");
                $("." + this_class).addClass("active").siblings().removeClass("active");
            });

            $content.children().each(function(index){
                $(this).addClass("onglet-" + index);
            });
        });
        
        $(window).load(function(){  window.setTimeout(resizeBloc, 700); });

        function resizeBloc() {

            $bloc300Liste = $("#blocResize");
            $bloc300ListeUl = $bloc300Liste.find("ul");

            $bloc300ListeUl.removeAttr('style');

            hauteur1erArticle = $("#premierArticle").outerHeight(true);
            hauteurColArticle = $("#colArticle").outerHeight(true);
            hauteurColDroite = $bloc300Liste.parents($("#colDroite")).outerHeight(true);

            if ( ($("#premierArticle").length) && ($("#premierArticle").hasClass('small')) ){
                dif = (hauteur1erArticle + hauteurColArticle) - hauteurColDroite; 
            } else {
                 dif = hauteurColArticle - hauteurColDroite;
            }

            if(dif < 0){
                dif = -1 * dif;
                var hauteurItem = $bloc300ListeUl.children("li").outerHeight();
                //items à rajouter ou enlever, arrondi à l'inférieur
                // si on souhaite arrondir au supérieur, il faut utiliser Math.ceil        
                var nbItems = Math.floor(dif / hauteurItem);

                $bloc300ListeUl.height($bloc300ListeUl.outerHeight() - nbItems * hauteurItem - 5);
            }
            
        }

        // On déclenche un événement scroll pour mettre à jour le positionnement au chargement de la page

                $toolBar.removeClass('fixed');
                $toolBar.css({"margin-top": positionTop, 'bottom': 'auto'});

        });
