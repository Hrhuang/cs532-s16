            <meta property="bt:body" content="Amazon has a surprise: it's built a free, open-source 3D engine for game developers called Lumberyard. Built for &quot;AAA&quot; games, the engine integrates the company's AWS cloud platform deeply to provide server-side processing.

Lumberyard is a full game studio; you can build and design levels, create your visuals, animate characters, generate particles, engineer audio, handle weather and AI and basically everything else you can imagine. It provides both the infrastructure for the multiplayer component of the game as well as the tools to build the game itself.

The game engine uses Cryengine for graphics, which can generate 'photo-realistic' worlds -- it's best known for being the engine behind the PC-crippling Crysis games.

A new AWS service called GameLift is integrated into the app to provide scalable multiplayer architecture without requiring deep infrastructure skills. The company touts that using GameLift means developers don't need to spend &quot;thousands of hours&quot; building out this infrastructure, instead promising it takes just a matter of minutes.

The new platform also offers deep Twitch integration, so developers can build in impressive new features to games. Twitch ChatPlay developers can build features that allow observers to join in with game streams by voting on levels or the game's direction and Twitch JoinIn lets broadcasters instantly invite viewers to join them in-game.

It's obvious why the company is making Lumberyard available for free -- as Microsoft has leveraged Azure for Xbox Live, Amazon wants game developers to use its cloud as well.

By building a tool like Lumberyard and making it freely available, the company is actively encouraging both developers and designers to deeply integrate Amazon's cloud stack into their titles; ultimately, it's locking them in long-term.

The most impressive part of Lumberyard is that it's both entirely free and the code is available for developers to modify. Other game engines can cost developers in the thousands and often charge for per-seat pricing, so Amazon's proposition is likely interesting for both large and small game developers.

                        <div class="l-postSingle-content-main-content postContent">
                            <a target="_blank" href="http://thenextweb.com/conference/europe/?utm_medium=display&amp;utm_source=thenextweb.com&amp;utm_campaign=TNWEU16_Yellowlink"><div class="announcement" style="color: #444444; cursor:pointer; padding-top: 10px; padding-bottom: 10px; margin-bottom: 1rem;"><span style="font-weight:bold">
                                      5 Days left</span>: TNW Europe early bird sales end soon. <span style="color:#FF4422;">Learn more!</span></div></a><p>Amazon has a surprise: it’s built a free, open-source 3D engine for game developers called <a href="http://aws.amazon.com/lumberyard/details/" target="_blank">Lumberyard</a>. Built for “AAA” games, the engine integrates the company’s AWS cloud platform deeply to provide server-side processing.</p>
<p>Lumberyard is a full game studio; you can build and design levels, create your visuals, animate characters, generate particles, engineer audio, handle weather and AI and basically everything else you can imagine. It provides both the infrastructure for the multiplayer component of the game as well as the tools to build the game itself.</p>
<p>The game engine <a href="https://docs.aws.amazon.com/lumberyard/latest/developerguide/graphics-rendering-intro.html" target="_blank">uses Cryengine</a> for graphics, which can generate ‘photo-realistic’ worlds — it’s best known for being the engine behind the PC-crippling <a href="http://crysis.com/" target="_blank">Crysis games</a>.</p>
<p>A <a href="http://aws.amazon.com/gamelift/" target="_blank">new AWS service called GameLift</a> is integrated into the app to provide scalable multiplayer architecture without requiring deep infrastructure skills. The company touts that using GameLift means developers don’t need to spend “thousands of hours” building out this infrastructure, instead promising it takes just a matter of minutes.</p>
<p>The new platform also offers deep Twitch integration, so developers can build in impressive new features to games. <a href="https://docs.aws.amazon.com/lumberyard/latest/userguide/chatplay-intro.html" target="_blank">Twitch ChatPlay</a> developers can build features that allow observers to join in with game streams by voting on levels or the game’s direction and <a href="https://docs.aws.amazon.com/lumberyard/latest/userguide/chatplay-joinin.html" target="_blank">Twitch JoinIn</a> lets broadcasters instantly invite viewers to join them in-game.</p>
<p>It’s obvious why the company is making Lumberyard available for free — as Microsoft has leveraged Azure for Xbox Live, Amazon wants game developers to use its cloud as well.</p>
<p>By building a tool like Lumberyard and making it freely available, the company is actively encouraging both developers and designers to deeply integrate Amazon’s cloud stack into their titles; ultimately, it’s locking them in long-term.</p>
<p>The most impressive part of Lumberyard is that it’s both entirely free and the code is available for developers to modify. Other game engines can cost developers in the thousands and often charge for per-seat pricing, so Amazon’s proposition is likely interesting for both large and small game developers.</p>
